# 🌍 Wanderspace (Travel Platform)

A backend-focused travel listing platform where users can explore, create, edit, and manage travel destinations. Built using **Node.js**, **Express**, and **MongoDB**, with server-side rendering using **EJS**.

## 🚀 Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS
- Bootstrap
- REST APIs

## 🔧 Features
- Full CRUD functionality for travel listings
- Add/Edit/Delete destinations
- Server-rendered pages using EJS
- Image support for travel locations
- Responsive UI using Bootstrap
- Modular route architecture
